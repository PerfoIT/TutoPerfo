Nextcloud — Informations utilisateur et signature mail

Tutoriel : Compléter ses informations Nextcloud

Ces informations servent à générer automatiquement votre signature mail PERFO.

Une information incorrecte dans Nextcloud peut générer une signature mail incorrecte.

1. Pourquoi compléter ces informations ?

La signature mail PERFO est générée automatiquement à partir des informations renseignées dans votre profil Nextcloud.

Il est donc important que chaque salarié vérifie et complète correctement son profil.

2. Accéder aux paramètres personnels

  1. Connectez-vous à AppPerfo / Authentik.
  2. Ouvrez l’application Nextcloud.
  3. Cliquez sur votre avatar ou vos initiales en haut à droite.
  4. Cliquez sur Paramètres personnels.
  5. Ouvrez la rubrique Informations personnelles ou Profil.

3. Informations à vérifier

ChampConsigne
Nom completPrénom et nom correctement orthographiés.
Adresse mailAdresse mail professionnelle PERFO.
Fonction / posteIntitulé utilisé dans votre signature mail.
Téléphone professionnelNuméro affiché dans la signature, si applicable.

Organisation / servicePERFO ou service concerné selon votre cas.
Respectez les majuscules, accents et espaces : ces informations sont reprises automatiquement.

4. Enregistrer les modifications

  1. Corrigez les informations nécessaires.
  2. Enregistrez les modifications si Nextcloud le demande.
  3. Déconnectez-vous puis reconnectez-vous si les informations ne se mettent pas à jour immédiatement.
  4. Patientez si la signature n’est pas modifiée instantanément : une synchronisation peut être nécessaire.

5. Points de contrôle

  • Votre prénom et votre nom sont correctement écrits.
  • Votre fonction est claire et professionnelle.
  • Votre numéro de téléphone est exact.
  • Aucune information personnelle inutile n’est renseignée.
