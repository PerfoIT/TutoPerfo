Tutoriel : Créer un mot de passe d’application mail

Configuration d’un compte mail dans une application externe.

À utiliser uniquement si vous souhaitez ajouter votre compte PERFO dans Outlook, Thunderbird, Apple Mail, Gmail mobile ou une autre application.

Pour utiliser le portail webmail depuis AppPerfo, vous n’avez pas besoin de mot de passe d’application.

1. À quoi sert le mot de passe d’application ?

Le mot de passe d’application sert à connecter un logiciel de messagerie externe à votre compte mail PERFO.

  • Il est différent de votre mot de passe AppPerfo.
  • Il peut être supprimé si un appareil est perdu ou remplacé.
  • Il évite de saisir votre mot de passe principal dans plusieurs applications.

2. Créer le mot de passe

  1. Connectez-vous à AppPerfo.
  2. Cliquez sur l’icône Mail pour ouvrir le portail webmail.
  3. Sur la page du portail webmail, cliquez sur Single Sign On.
  4. Ouvrez les paramètres de votre compte mail.
  5. Accédez à la rubrique liée aux mots de passe d’application ou aux accès externes.
  6. Créez un nouveau mot de passe avec un nom explicite, par exemple Outlook PC portable ou iPhone.
  7. Copiez le mot de passe généré et conservez-le temporairement le temps de configurer votre application.
Le mot de passe généré peut ne plus être visible ensuite. Copiez-le immédiatement.

3. Configuration dans l’application de messagerie

Dans la majorité des cas, la configuration se fait automatiquement grâce à l’autodiscover. Il suffit généralement de saisir votre adresse mail PERFO et le mot de passe d’application généré.

Les paramètres ci-dessous sont à renseigner uniquement si votre application de messagerie les demande ou si la détection automatique échoue.

4. Bonnes pratiques

  • Créez un mot de passe différent par appareil ou application.
  • Supprimez les mots de passe inutilisés.
  • Ne partagez jamais un mot de passe d’application.
  • En cas de doute, supprimez-le et recréez-en un nouveau.
