Tutoriel : Créer un mot de passe d’application mail

À utiliser uniquement si vous souhaitez ajouter votre compte PERFO dans Outlook, Thunderbird, Apple Mail, Gmail mobile ou une autre application.

Pour utiliser SOGo depuis AppPerfo, vous n’avez pas besoin de mot de passe d’application.

1. À quoi sert le mot de passe d’application ?

Le mot de passe d’application sert à connecter un logiciel de messagerie externe à votre compte mail PERFO.

  • Il est différent de votre mot de passe Authentik.
  • Il peut être supprimé si un appareil est perdu ou remplacé.
  • Il évite de saisir votre mot de passe principal dans plusieurs applications.

2. Créer le mot de passe

  1. Connectez-vous à AppPerfo / Authentik.
  2. Cliquez sur l’icône Mail pour ouvrir SOGo.
  3. Ouvrez les paramètres de votre compte mail.
  4. Accédez à la rubrique liée aux mots de passe d’application ou aux accès externes.
  5. Créez un nouveau mot de passe avec un nom explicite, par exemple Outlook PC portable ou iPhone.
  6. Copiez le mot de passe généré et conservez-le temporairement le temps de configurer votre application.
Le mot de passe généré peut ne plus être visible ensuite. Copiez-le immédiatement.

3. Paramètres de configuration mail

Dans votre application de messagerie, utilisez les paramètres suivants :

ParamètreValeur
Adresse mailvotre.adresse@perfo-formation.fr
Nom d’utilisateurVotre adresse mail PERFO complète
Mot de passeLe mot de passe d’application généré
Serveur entrant IMAPmail.perfo-formation.fr
Port IMAP993 — SSL/TLS
Serveur sortant SMTPmail.perfo-formation.fr
Port SMTP recommandé465 — SSL/TLS
Port SMTP alternatif587 — STARTTLS

4. Bonnes pratiques

  • Créez un mot de passe différent par appareil ou application.
  • Supprimez les mots de passe inutilisés.
  • Ne partagez jamais un mot de passe d’application.
  • En cas de doute, supprimez-le et recréez-en un nouveau.
