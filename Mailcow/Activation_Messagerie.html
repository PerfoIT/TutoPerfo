Tutoriel : Activer sa nouvelle messagerie PERFO

Ce guide explique comment accéder à votre nouvelle boîte mail PERFO depuis AppPerfo / Authentik.

Action obligatoire : vous devez cliquer au moins une fois sur l’icône Mail dans Authentik pour créer et activer votre boîte mail.

1. Ce qui change

La messagerie PERFO passe sur une nouvelle architecture basée sur Mailcow, avec une connexion centralisée via Authentik.

Avant
Connexion avec votre adresse mail personnelle.
Maintenant
Connexion avec votre adresse mail PERFO ou votre nom d’utilisateur.
Création automatique
Votre boîte mail est créée automatiquement au premier clic sur l’icône Mail.

2. Se connecter à AppPerfo / Authentik

Lors du changement, votre connexion à Authentik pourra se faire avec l’un des deux identifiants suivants :

MéthodeExempleRemarque
Nouvelle adresse mail PERFO prenom.nom@perfo-formation.fr À utiliser dès que votre nouvelle adresse vous a été communiquée.
Nom d’utilisateur qmarques Première lettre du prénom + nom, tout attaché, sans accent ni espace.
Exemple : Quentin Marques devient qmarques.

3. Activer la boîte mail

  1. Ouvrez la plateforme AppPerfo / Authentik.
  2. Connectez-vous avec votre nouvelle adresse mail PERFO ou votre nom d’utilisateur.
  3. Sur l’écran des applications, cliquez sur l’icône Mail.
  4. Patientez pendant l’ouverture de SOGo.
  5. Une fois SOGo ouvert, votre boîte mail est créée et activée.
Ne sautez pas cette étape : sans premier clic sur Mail, votre boîte mail peut ne pas être créée.

4. Utiliser la messagerie avec SOGo

Après activation, vous pouvez consulter vos mails directement depuis SOGo, accessible via l’icône Mail dans AppPerfo.

  • Vos mails sont accessibles depuis un navigateur web.
  • La connexion se fait via le SSO Authentik.
  • Vous n’avez pas besoin de configurer manuellement un logiciel de messagerie si vous utilisez uniquement SOGo.

5. Après l’activation

Deux actions complémentaires peuvent être nécessaires :

Configurer Outlook, Thunderbird ou un téléphone
Créez un mot de passe d’application spécifique pour votre logiciel de messagerie.
Compléter vos informations Nextcloud
Ces informations servent à générer automatiquement votre signature mail.
Créer un mot de passe d’application Compléter mes informations Nextcloud
