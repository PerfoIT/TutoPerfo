AppPerfo & Nextcloud

Tutoriel : Accès à la Plateforme AppPerfo et Nextcloud

Guide détaillé pour configurer votre compte et utiliser les services

Étape 1 : Réception de l'Invitation par E-mail

  1. Ouvrez votre boîte de réception e-mail. Assurez-vous également de vérifier vos spams si vous ne voyez pas immédiatement le message.
  2. Recherchez un e-mail intitulé "Invitation à rejoindre la plateforme AppPerfo". Cet e-mail provient de notre service informatique.
  3. Ouvrez l'e-mail. Vous y trouverez un texte d'introduction expliquant que vous avez été invité à rejoindre la plateforme.
  4. Cliquez sur le bouton ou le lien intitulé "Accepter l'invitation" ou similaire. Ce lien vous redirigera vers une page sécurisée où vous pourrez configurer votre compte.

Étape 2 : Création de Votre Compte

  1. Après avoir cliqué sur le lien dans l'e-mail, une nouvelle page s'ouvre dans votre navigateur.
  2. Dans le champ "Prénom", entrez votre prénom
  3. Dans le champ "Nom", entrez votre nom de famille
  4. Dans le champ "Email", entrez votre email professionnel
  5. Dans le champ "Mot de passe", choisissez un mot de passe fort en suivant ces règles :
    • Utilisez au moins 12 caractères.
    • Incluez des lettres majuscules (A-Z), minuscules (a-z), chiffres (0-9) et caractères spéciaux (@#$%^&*()).
    • Évitez les mots courants, les séquences simples comme "123456" ou des informations personnelles faciles à deviner (comme votre date de naissance).
    • Exemple de mot de passe fort : T3chN0l0gie@2023.
  6. Dans le champ "Confirmer le mot de passe", ressaisissez exactement le même mot de passe que vous venez de créer.
  7. Cliquez sur le bouton "Continuer" pour valider votre inscription.
  8. Vous verrez un message de confirmation indiquant que votre compte a été créé avec succès.
Important : Ne partagez jamais votre mot de passe avec quiconque, même si on prétend être un administrateur système.

Étape 3 : Connexion à AppPerfo

    Suite à votre inscription, vous devriez automatiquement être redirigé vers la page d'acceuil de AppPerfo (et donc, déja connecté), mais si ce n'est pas le cas, vous pouvez suivre les étapes ci-dessous.

  1. Ouvrez votre navigateur web (Google Chrome, Mozilla Firefox, etc.).
  2. Dans la barre d'adresse, saisissez l'URL (ou cliquez sur ce lien) : https://app.perfo.fr
  3. Dans le champ "Nom d'utilisateur", entrez le nom d'utilisateur ou votre email que vous avez choisi lors de la configuration de votre compte.
  4. Dans le champ "Mot de passe", saisissez le mot de passe que vous avez créé.
  5. Cliquez sur le bouton "Se connecter".
  6. Si vos informations sont correctes, vous serez redirigé vers le tableau de bord.

Étape 4 : Accéder à Nextcloud via SSO

  1. Une fois connecté à AppPerfo, observez l'interface principale. Vous verrez plusieurs icônes représentant les services disponibles.
  2. Cherchez l'icône aec un nuage bleu appelé "Cloud".
  3. Cliquez sur cette icône. Grâce au système de Single Sign-On (SSO), vous serez automatiquement authentifié et redirigé vers votre espace personnel sur Nextcloud.
  4. Explorez votre espace Nextcloud :
    • Téléversez des fichiers en cliquant sur le bouton "Ajouter un fichier".
    • Téléchargez des fichiers en cliquant sur leur nom ou l'icône correspondante.
    • Partagez des dossiers ou fichiers avec vos collègues en utilisant les options de partage disponibles.

Conseils Importants

  • Mettez régulièrement à jour votre mot de passe : Changez votre mot de passe tous les 3 à 6 mois pour maintenir un niveau élevé de sécurité.
  • Activez l'authentification à deux facteurs (2FA) : Si cette option est disponible dans AppPerfo, configurez-la dès que possible. Cela ajoute une couche de sécurité supplémentaire en demandant un code unique généré par une application mobile (par exemple Google Authenticator) à chaque connexion.
  • Signalez tout problème immédiatement : Si vous rencontrez des difficultés pour accéder à la plateforme ou si vous suspectez que votre compte a été compromis, contactez immédiatement l'équipe informatique.
Pour toute assistance, contactez le support informatique PERFO.
