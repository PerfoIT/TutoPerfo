Tutoriel : Installation et Paramétrage de Nextcloud Talk

Guide détaillé pour installer Nextcloud Talk sur Android, iOS et PC, et activer les notifications

📋 Prérequis

  • Un compte utilisateur sur votre serveur Nextcloud (https://cloud.perfo.fr)
  • L'application Talk activée par l'administrateur sur le serveur
  • Une connexion Internet stable
  • Pour les notifications push mobiles : services Google Play (Android) ou compte Apple (iOS)
⚠️ Important : Les notifications push sur Android nécessitent les services Google Play. La version F-Droid de l'application ne prend pas en charge les notifications push [[24]].

📱 Installation sur Android Android

Étape 1 : Télécharger l'application

  1. Ouvrez le Google Play Store sur votre appareil Android ou cliquer sur le bouton ci-dessous.
  2. Recherchez "Nextcloud Talk".
  3. Téléchargez l'application officielle développée par Nextcloud (icône bleue avec bulle de dialogue).
  4. Appuyez sur "Installer" et acceptez les permissions demandées.
💡 Alternative : Vous pouvez également installer l'application via F-Droid, mais sachez que les notifications push ne fonctionneront pas sans les services Google Play [[24]].

Étape 2 : Configurer la connexion au serveur

  1. Ouvrez l'application Nextcloud Talk.
  2. Dans le champ "Adresse du serveur", saisissez l'URL de votre instance : https://cloud.perfo.fr
  3. Appuyez sur "Suivant".
  4. Vous serez redirigé vers la page de connexion SSO App Perfo :
    • Entrez votre nom d'utilisateur ou adresse e-mail
    • Entrez votre mot de passe
    • Si l'authentification à deux facteurs (2FA) est activée, saisissez le code généré
  5. Après authentification réussie, autorisez l'accès de l'application à votre compte Nextcloud.

Étape 3 : Activer les notifications push

  1. Dans l'application Talk, appuyez sur l'icône ☰ Menu (en haut à gauche).
  2. Allez dans Paramètres → Notifications.
  3. Activez les options suivantes :
    • ✅ Notifications de conversation
    • ✅ Notifications d'appel
    • ✅ Son et vibration (selon vos préférences)
  4. Assurez-vous que les notifications de l'application ne sont pas bloquées au niveau du système Android :
    • Allez dans Paramètres Android → Applications → Nextcloud Talk → Notifications
    • Vérifiez que toutes les catégories de notifications sont activées
🔧 Dépannage notifications Android :
Si vous ne recevez pas de notifications :
• Vérifiez que les services Google Play sont à jour
• Désactivez l'optimisation de batterie pour Nextcloud Talk
• Redémarrez l'application et votre appareil
• Consultez la checklist officielle de dépannage [[24]]

🍎 Installation sur iOS iOS

Étape 1 : Télécharger l'application

  1. Ouvrez l'App Store sur votre iPhone ou iPad.
  2. Recherchez "Nextcloud Talk".
  3. Téléchargez l'application officielle (éditeur : Nextcloud).
  4. Appuyez sur "Obtenir" et authentifiez-vous avec Face ID, Touch ID ou votre identifiant Apple.

Étape 2 : Configurer la connexion au serveur

  1. Ouvrez l'application Nextcloud Talk.
  2. Tapez sur "Ajouter un serveur".
  3. Saisissez l'URL de votre instance : https://cloud.perfo.fr
  4. Appuyez sur "Suivant".
  5. Connectez-vous via la page SSO App Perfo :
    • Entrez vos identifiants App Perfo
    • Validez l'authentification à deux facteurs si activée
  6. Autorisez l'application à accéder à votre compte Nextcloud.

Étape 3 : Activer les notifications push

  1. Lors de la première utilisation, iOS vous demandera l'autorisation d'envoyer des notifications : appuyez sur "Autoriser".
  2. Si vous avez refusé par erreur, allez dans :
    • Réglages iOS → Notifications → Nextcloud Talk
    • Activez "Autoriser les notifications"
    • Cochez les options : Bannières, Sons, Badges
  3. Dans l'application Talk : Paramètres → Notifications :
    • ✅ Activer les notifications de messages
    • ✅ Activer les notifications d'appels entrants
    • ✅ Personnaliser les sons et vibrations
🔔 Astuce iOS : Pour garantir la réception des notifications en arrière-plan, assurez-vous que l'application n'est pas fermée manuellement (ne la "swipez" pas vers le haut dans le multitâche) [[2]].

💻 Installation sur PC Desktop

Étape 1 : Télécharger le client Talk Desktop

Nextcloud Talk propose un client desktop dédié pour Windows, macOS et Linux [[5]].

Système Lien de téléchargement Méthode alternative
Windows GitHub Releases winget install Nextcloud.Talk [[1]]
macOS GitHub Releases (Universal) Homebrew (à venir)
Linux AppImage / Flatpak Flatpak : flatpak install flathub com.nextcloud.Talk

Étape 2 : Installation

  • Windows : Exécutez le fichier .exe téléchargé et suivez l'assistant d'installation [[6]].
  • macOS : Ouvrez le fichier .dmg, glissez l'application dans le dossier Applications.
  • Linux :
    • AppImage : Rendez le fichier exécutable (chmod +x) puis lancez-le
    • Flatpak : Installez via la commande indiquée ci-dessus

Étape 3 : Configuration et connexion

  1. Lancez Nextcloud Talk Desktop.
  2. Dans le champ "Server URL", entrez : https://cloud.perfo.fr
  3. Cliquez sur "Connect".
  4. Une fenêtre de navigateur s'ouvre pour l'authentification SSO via App Perfo :
    • Connectez-vous avec vos identifiants
    • Validez l'accès pour l'application Talk
  5. De retour dans l'application, vos conversations apparaissent.

Étape 4 : Activer les notifications desktop

  1. Dans Talk Desktop, cliquez sur l'icône ⚙️ Paramètres (en bas à gauche).
  2. Allez dans l'onglet Notifications.
  3. Activez les options :
    • ✅ Afficher les notifications de bureau
    • ✅ Jouer un son pour les nouveaux messages
    • ✅ Afficher un badge sur l'icône
  4. Vérifiez les paramètres système :
    • Windows : Paramètres → Système → Notifications → Nextcloud Talk → Activé
    • macOS : Préférences Système → Notifications → Nextcloud Talk → Autoriser
    • Linux : Paramètres de votre environnement de bureau (GNOME/KDE) → Notifications
⚠️ Note importante : Sur Linux, les notifications nécessitent un serveur de notifications compatible (libnotify). Assurez-vous que votre environnement de bureau supporte les notifications desktop.

⚙️ Configuration avancée des notifications

Personnalisation par conversation

Dans chaque conversation Talk (mobile ou desktop) :

  1. Ouvrez la conversation souhaitée.
  2. Appuyez sur le nom de la conversation ou l'icône ⋮ / ⋯.
  3. Allez dans Paramètres de notification.
  4. Choisissez le niveau de notification :
    • 🔔 Tous les messages : Notification à chaque message
    • 🔕 Mentions uniquement : Notification seulement si vous êtes mentionné (@vous)
    • 🚫 Aucune : Désactiver les notifications pour cette conversation

Mode "Ne pas déranger"

  • Sur mobile : Utilisez le mode "Ne pas déranger" du système pour suspendre temporairement toutes les notifications.
  • Dans Talk : Vous pouvez définir des plages horaires silencieuses dans Paramètres → Notifications → Planification (si disponible sur votre version).

🔧 Dépannage des notifications

Problème Solution
Aucune notification reçue sur Android • Vérifier que Google Play Services est actif
• Désactiver l'optimisation batterie pour Talk
• Vérifier les permissions système [[24]]
Notifications iOS retardées • Vérifier la connexion Internet
• S'assurer que l'app n'est pas fermée manuellement
• Redémarrer l'appareil
Notifications desktop non affichées • Vérifier les paramètres de notification du système d'exploitation
• S'assurer que Talk n'est pas en mode "silencieux"
• Redémarrer l'application
Erreur de connexion au serveur • Vérifier l'URL du serveur (https://cloud.perfo.fr)
• Confirmer que le module Talk est activé côté serveur
• Contacter l'administrateur
🌐 Documentation officielle : Pour plus d'informations, consultez la documentation utilisateur Nextcloud Talk : docs.nextcloud.com [[18]]

💡 Conseils d'utilisation

  • Gardez les applications à jour : Les mises à jour apportent des corrections de sécurité et de nouvelles fonctionnalités.
  • Utilisez un mot de passe d'application : Pour plus de sécurité, générez un mot de passe dédié dans Nextcloud Web → Paramètres → Sécurité.
  • Testez vos notifications : Après configuration, demandez à un collègue de vous envoyer un message test pour vérifier la réception.
  • Respectez la vie privée : Nextcloud Talk étant auto-hébergé, vos données de communication restent sous votre contrôle.
